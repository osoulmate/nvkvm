#### 介绍
nvkvm(No Vendor kvm)跨厂商的ipmi远程控制台应用
#### 支持机型
- 华为服务器全系（存储机型除外）
- 联想System X系列
- 新华三[uniserver r4900g3,uniserver r4700g3]
- 曙光[i640-g15,i620-g20,i620-g30,w780-g20,w580-g20]
- 烽火[fitserver系列]
- 浪潮[NF5288M5,NF5270M3,NF8420M3]
- 中兴[r5300g4,]
- 惠普
- 戴尔
