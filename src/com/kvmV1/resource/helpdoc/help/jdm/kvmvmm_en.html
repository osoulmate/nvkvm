Context

You can remotely monitor and manage servers, install and repair OSs, and install device drivers by using the following methods:

  • Use the local keyboard, video, and mouse (KVM) on the client to monitor and control remote devices in real time.
  • Use a server to access the local media on the client (the host of a customer) over a network in the mode of the virtual floppy disk drive (FDD) and virtual CD-ROM drive. From the view of the server, the operation method of the virtual FDD or virtual CD-ROM drive is the same as that of the physically inserted USB device.
    NOTE:
    The local media on the client can be an FDD, a CD-ROM drive, or a floppy disk image file or CD-ROM image file stored on the client or on the network drive.

Click . The displayed menu contains Power On, Power Off, Cold Reset, Graceful Reboot, USB Reset, Video.

  • Power management involves powering on, powering off, cold reset, and graceful reboot.

    NOTE:

    • Power On: powers on a device.

    • Power Off: powers off a device.

    • Code Reset: performs cold reset on a device.

    • Graceful Reboot: performs soft reset on the device, that is, performs log-off only.

  • USB Reset: resets the USB keyboard and mouse.

  • Video: records a video for the current remote control screen.

    NOTE:

    The video file is in .rep format. You can play or edit it by choosing Diagnose and Location > Video Play.
